Description - print_sorted(self,)

Public instance method that prints the list, but sorted (ascending sort)

------------------------------------------------------------------------

-------

    >>> MyList = __import__('1-my_list').MyList


------------------
Successful test
------------------

append module add integers to MyList:

    >>> my_list.append(1)
    >>> my_list.append(34)
    >>> my_list.print_sorted()
